## 📌 About the Project
INNOVATRIX is a platform designed to bridge the gap between **education and employability** by allowing users to showcase their real-world skills through:

- Verified projects  
- Skill-based assessments  
- Peer validation  

---

## 🎯 Problem We Are Solving
Traditional resumes don’t reflect actual skills.  
INNOVATRIX focuses on **proof of work**, not just claims.
